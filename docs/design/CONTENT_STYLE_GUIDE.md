# Content Style Guide

## 1. Voice

YAGI Workspace copy should be calm, precise, and helpful.

It should feel like a professional creative studio, not a generic SaaS assistant.

## 2. Korean-first

Default UI language is Korean.

Use concise Korean labels.

Good:
- 기획 보드
- 레퍼런스 추가
- 파일 업로드
- 새 버전 저장
- 승인 요청
- 제출물 검토
- 최종본 확정
- 변경사항 저장됨

Avoid:
- 지금 시작하기
- 창의적인 여정을 시작하세요
- AI로 마법 같은 결과를 만들어보세요
- 당신의 생산성을 혁신하세요

## 3. Button copy

Use action verbs.

Good:
- 저장
- 업로드
- 승인 요청
- 검토하기
- 제출하기
- 보드에 추가

Avoid long phrases:
- 이 프로젝트의 기획 내용을 확인하러 가기
- 제출된 작업물을 자세히 검토하러 이동하기

## 4. Status labels

Project:
- 기획
- 제작
- 리뷰
- 수정
- 완료

Contest:
- 초안
- 검토 중
- 예정
- 진행 중
- 마감
- 심사 중
- 수상작 선정
- 보관됨

Submission:
- 작성 중
- 제출됨
- 검토 중
- 수정 요청
- 통과
- 반려
- 파이널리스트
- 수상

## 5. AI labels

Use restrained language.

Good:
- 검토할 제안
- 기획 보완점
- 빠진 정보
- 보드에 추가
- 제안 닫기

Avoid:
- AI가 완성
- 마법처럼 생성
- 자동으로 해결
- 완벽한 기획서

## 6. Error messages

Explain what happened and the next action.

Good:
- 파일 업로드에 실패했습니다. 네트워크 상태를 확인한 뒤 다시 시도해주세요.
- 이 프로젝트에 접근할 권한이 없습니다.
- 이미 승인된 버전은 수정할 수 없습니다. 새 버전으로 업로드해주세요.

Avoid:
- 오류가 발생했습니다.
- 실패했습니다.
