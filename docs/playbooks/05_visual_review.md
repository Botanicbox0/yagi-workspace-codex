# 05 Visual Review

Use for design-facing gates and UI changes.

## Agent

Design Director.

## Required Reading

- `docs/agents/DESIGN_DIRECTOR.md`
- `docs/design/DESIGN_SYSTEM.md`
- `docs/design/VISUAL_QA_CHECKLIST.md`
- `docs/design/CONTENT_STYLE_GUIDE.md`
- `docs/product/INFORMATION_ARCHITECTURE.md`

## Steps

1. Review changed screens and components.
2. Check surface separation: private app, challenge, creator, showcase.
3. Check Korean copy and hierarchy.
4. Check forbidden visual patterns.
5. Check focus states, empty states, and density.
6. Record findings with severity and file/path.
