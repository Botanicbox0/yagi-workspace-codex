# ADR 0006 — Design Direction

Status: Accepted

## Context

AI SaaS products often look generic: purple gradients, glow, glass cards, emoji-heavy UI, and default component styling.

YAGI Workspace must feel premium and studio-grade.

## Decision

Adopt the design direction:

> A private creative operations room: quiet, editorial, precise, Korean-first, visual-first.

Use Contact.xyz only as a reference for editorial confidence in creator/contest/showcase surfaces, not as a marketplace model.

## Consequences

- Private workspace uses calm Studio OS language.
- Contest/creator/showcase can be more editorial.
- Generic AI SaaS patterns are forbidden.
- Design QA is required before merge.
